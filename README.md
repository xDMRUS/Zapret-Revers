Реверсивная версия Zapret работает только с YouTube и Discord, не затрагивает другие процессы и не мешает подключению к онлайн-играм.

- Установка аналогична обычной версии:

1. Запустите service.bat.

2. Выберите Install Service.

3. Устанавливаем нужный (bat-файл).

4. Готово

- YouTube и Discord работают.
